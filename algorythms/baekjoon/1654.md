## 오답
```python
K,N = map(int,input().split())
arr=[]

for i in range(K):
    arr.append(int(input()))

sumArr = sum(arr)
arr.sort()
# answer <arr[0] and answer <= sumArr//11
# the greast = sumArr//11
answer = sumArr//N
while True :
    cnt = 0
    for i in arr :
        cnt +=i//answer
    if cnt >= N :
        break
    else :
        answer -=1

print(answer)
```
## 정답
```python
K,N = map(int,input().split())
arr=[]

for i in range(K):
    arr.append(int(input()))

start= 1
end = max(arr)

while start <= end:
	cnt = 0
	mid = (start + end)//2
	for i in arr :
		cnt += i//mid
	if cnt >=N :
		start = mid+1
	else :
		end = mid -1
print(end)
```
